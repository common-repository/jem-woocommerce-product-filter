== Description ==
WooCommerce Product Filter is a super-easy to use plugin that allows you to add filters quickly and easily to your WooCommerce store. With this plugin your customers can quickly and easily find the products they want to purchase. 

It comes with a ton of great features including:

= Features =

* Filter by Price using a slider
* Filter by attribute
* Filter by category
* AJAX filterung, pagination & sorting
* SEO friendly URL's - keep your SEO juice!
* We are constantly adding new features and functionality! Let us know what you need!

== Installation ==


1. Upload `jem-woocommerce-product-filter.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can now add the filter thru the widget
